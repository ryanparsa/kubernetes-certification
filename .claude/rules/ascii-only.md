# ASCII-Only Policy

Do **not** generate non-ASCII characters (e.g., emojis, smart quotes, em dashes) in this repository.

Always use the following ASCII representations instead of their emoji counterparts:
- `📋` -> `[CODE]`
- `📖` -> `[CONCEPT]`
- `✅` -> `[OK]`
- `❌` -> `[FAIL]`
- `⚠` -> `[WARN]`
- `⚡` -> `[TIP]`
- `—` -> `--`
- `→` -> `->`

The only exception is Box Drawing characters (U+2500 - U+253C), which are permitted exclusively for directory tree diagrams in the `ref/` docs.
