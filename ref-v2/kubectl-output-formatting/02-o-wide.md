## 2. `-o wide`

```bash
kubectl get pods -o wide
# NAMESPACE  NAME     READY  STATUS   RESTARTS  AGE  IP           NODE       ...

kubectl get nodes -o wide
# NAME      STATUS  ROLES    AGE  VERSION  INTERNAL-IP  EXTERNAL-IP  OS-IMAGE  ...
```
