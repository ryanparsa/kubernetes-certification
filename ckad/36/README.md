> **Solve this question on:** `ckad-lab-36`

For improved container image distribution and portability, you need to work with the Open Container Initiative (OCI) format.

Complete the following steps:

1. Pull the `nginx:latest` image from Docker Hub using the `docker pull` command

2. Create a directory at `/root/oci-images` if it doesn't already exist

3. Using the appropriate docker commands, export the nginx image in OCI format and store it in this directory

This will help standardize the container image format and improve portability across different container runtimes.
