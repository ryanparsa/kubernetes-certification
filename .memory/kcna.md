# KCNA Memory

KCNA content lives in `kcna/` and contains study material only — no lab/cluster structure.
Exam format: 60 multiple-choice questions, 90-minute limit, 75% passing score, $250 (one free retake).
Exam domains (2025/2026): Kubernetes Fundamentals 44-46%, Container Orchestration 22-28%, Cloud Native Architecture (incl. Observability) 12-16%, Cloud Native Application Delivery 8-16%.
"Cloud Native Observability" was merged into "Cloud Native Architecture" starting 2025.
Key files: `kcna-assessment-bank.md` (200 practice Qs), `kcna-exam-checklist.md` (domain checklist), `resources.md` (curated links).
