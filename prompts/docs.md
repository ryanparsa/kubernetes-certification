Practice type: docs navigation under exam constraints. Standalone -- no base or exam scope needed.

Allowed sites only: `kubernetes.io/docs`, `kubernetes.io/blog`, `helm.sh/docs`, `gateway-api.sigs.k8s.io`. No bookmarks, no other sites.

## Format

Each challenge is a realistic exam task that requires looking something up. I must answer with ALL of:

1. Which site
2. The exact search-bar term I'd type (favor this -- almost always faster than clicking)
3. The click-through navigation path as a fallback
4. What I'd copy from that page

## Grading

- Grade on speed and directness.
- [!] if I clicked when a search would have been faster -- show the better search term.
- Occasionally say "exam clock -- 30 seconds" and tell me afterward if my path was fast enough.
- Point out a nearby page I'd commonly need next, when relevant.

## Difficulty Ramp

Easy (cheat sheet, basic examples) -> medium (NetworkPolicy, TLS, backup/restore, RBAC) -> hard (encryption at rest, audit policy, JSONPath) -> expert (combine info from 2 doc pages).

Start with challenge #1.
